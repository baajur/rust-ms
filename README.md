# rust-microservice skeleton

:fishing_pole_and_fish: Secure High Performance Microservices

## Ships with

* Logger
* JSON Config Loader
* HTTP Server
* Redis Client
* HTTP Client
* ORM (MySQL) Client

## Setup Instructions

* `git clone https://github.com/krystianity/rust-ms.git`
* `cd rust-ms`
* `cargo build`
* `RUST_LOG=rust_ms cargo run`
